# 🎓 Vue.js Learning & Projects Journey

Bienvenue dans mon dépôt dédié à l'apprentissage de **Vue.js 3**. Ce repo retrace mon parcours, des concepts fondamentaux jusqu'à la réalisation de projets complets et complexes.

## 📂 Structure du Dépôt

Le dépôt est organisé par étapes progressives :

### [01-VUE-BASICS](file:///c:/Users/ndiay/OneDrive/Bureau/Vuejs/01-VUE-BASICS)
Introduction aux concepts de base de Vue.js :
- Syntaxe de template et directives (`v-bind`, `v-model`, `v-if`, `v-for`).
- Gestion des événements et propriétés calculées (`computed`).
- Introduction à la **Composition API**.

### [02-COMPONENTS](file:///c:/Users/ndiay/OneDrive/Bureau/Vuejs/02-COMPONENTS)
Maîtrise de l'architecture par composants :
- Props, Custom Events et Slots.
- Cycle de vie des composants.
- Réutilisabilité et patterns de communication.

### [03-ROUTING-STATE](file:///c:/Users/ndiay/OneDrive/Bureau/Vuejs/03-ROUTING-STATE)
Gestion du routage et de l'état global :
- Configuration de **Vue Router 4**.
- Gestion de l'état avec **Pinia**.
- Navigation Guards et protection des routes.

### [04-API-ASYNC](file:///c:/Users/ndiay/OneDrive/Bureau/Vuejs/04-API-ASYNC)
Interaction avec le monde réel :
- Appels API avec `fetch` et `axios`.
- Gestion des états de chargement et d'erreur.
- Cycle de vie asynchrone dans les composants.

### [05-MINI-PROJECT](file:///c:/Users/ndiay/OneDrive/Bureau/Vuejs/05-MINI-PROJECT)
Réalisation d'un projet d'envergure :
- **Admin Dashboard** : Une application complète de gestion e-commerce.
- Authentification, CRUD complet, Tableaux de bord avec **Chart.js**.
- Design Premium avec **Tailwind CSS v4** et Glassmorphism.

## 🛠️ Stack Technique Globale

- **Framework** : Vue 3 (Composition API & `<script setup>`)
- **Build Tool** : Vite
- **State Management** : Pinia
- **Styling** : Tailwind CSS v4 & Vanilla CSS
- **Icons** : Lucide Vue
- **Visualisation** : Chart.js

## 🚀 Comment explorer ce projet ?
